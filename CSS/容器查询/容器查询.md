# 容器查询

.container为父容器，名称为container，宽度为20vw，即随着视口宽度而缩放。

子div的初识颜色为灰色，当父容器宽度大于200px时，子div会变成浅蓝色。

当父容器的宽度大于300px时，子div会变成蓝色，同时宽度由固定值改为父容器宽度的80%。 

注意：容器的宽度变化只影响其子代和后代，其它的不受影响。 

改变浏览器窗口大小查看颜色变化的效果。
