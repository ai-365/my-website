var ​

var关键字的意思是，让编译器自己推断数据类型，而且一旦推断出来，那么类型不可改变，例如：

java
var a = 1;

等价于

java
int a = 1;

ArrayList ​

ArrayList 类是一个可以动态修改的数组，与普通数组的区别就是它是没有固定大小的限制，我们可以添加或删除元素。

java
ArrayList<String> sites = new ArrayList<String>();
        sites.add("Google");
        sites.add("Runoob");
        sites.add("Taobao");
        sites.add("Weibo");
        System.out.println(sites);
    }


[Google, Runoob, Taobao, Weibo]
