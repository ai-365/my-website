
# 映射

Python映射与字典类似，在有些时候，使用映射更加方便。

映射可以合并，后面的操作数中的元素如果覆盖前面的同名键，增加新的键。

使用 | 操作符更新映射。

```
d1 = {'a':1,'b':2}

# 键a的值会覆盖前面的
d2 = {'a':11,'c':3}

print(d1 | d2)
# {'a': 11, 'b': 2, 'c': 3}
# d1和d2都没有变
```

如果想就地更新映射，使用 |=。

```
d1 = {'a':1,'b':2}

# 键a的值会覆盖前面的
d2 = {'a':11,'c':3}

d1 |= d2

print(d1)
# {'a': 11, 'b': 2, 'c': 3}
```
